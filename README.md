Getting Started

Backend (Express)
Navigate to the backend folder:
cd backend

Install dependencies:
npm install

Start the backend server:
npm start


Frontend (Nuxt.js)
Navigate to the frontend folder:
cd frontend
cd parking-reservation

Install dependencies:
npm install

Running the App
Start the backend server:
cd backend
npm start

Start the frontend:
cd frontend
npm run dev
